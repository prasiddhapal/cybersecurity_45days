## Evaluation Criteria

- Password Length
- Uppercase Letters
- Lowercase Letters
- Numbers
- Special Characters
- Randomness
- Predictability

---

## Common Password Attacks

### Brute Force Attack
Attempts every possible combination until the correct password is found.

### Dictionary Attack
Uses a list of common words and leaked passwords.

### Credential Stuffing
Uses previously leaked username/password combinations on different websites.

---

## Best Practices

- Use at least 12–16 characters.
- Combine uppercase, lowercase, numbers, and symbols.
- Avoid personal information.
- Never reuse passwords.
- Use a password manager.
- Enable Multi-Factor Authentication (MFA).
